# Dados Geoespaciais

Coloque aqui os shapefiles compactados em .zip:

- `quadras.zip` — Quadras urbanas
- `lotes.zip` — Lotes / parcelas
- `logradouros.zip` — Logradouros (ruas, avenidas, etc.)

Cada .zip deve conter pelo menos os arquivos: `.shp`, `.dbf`, `.shx`
Opcionalmente: `.prj` (projeção) e `.cpg` (encoding)
